
# 2D Lighting
A technique for 2D lighting using a framebuffer

#### Dependencies  
* [GLFW] for window management
* [GLAD] for management of OpenGL extensions
* [GLM] for maths


[GLFW]: http://www.glfw.org/
[GLAD]: https://glad.dav1d.de/
[GLM]: http://glm.g-truc.net/0.9.7/index.html
